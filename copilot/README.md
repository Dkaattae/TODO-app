# vibe coding

## install django
`pip install django`

## create project
```
django-admin startproject myproject .
python manage.py startapp todo
```

## migrate
`python manage.py makemigrations`

## implement logics
use ai to generate code given prompt
see if implemented in views.py

## register template
copilot automatically registered it,   
see files in template

## test
ask copilot to add some test
run `python manage.py test`

## run
`python manage.py runserver`
